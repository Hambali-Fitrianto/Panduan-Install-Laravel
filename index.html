Panduan Install Laravel

Langkah-langkah instalasi Laravel untuk berbagai versi.

Persiapan

  • Pastikan PHP versi terbaru (minimal 8.1).
  • Install Composer dari getcomposer.org.
  • Siapkan database (MySQL, PostgreSQL, atau lainnya).

Langkah Instalasi Laravel

  1. Jalankan perintah berikut untuk menginstal Laravel: 
    composer create-project laravel/laravel:^11.0 nama-proyek
  2. Masuk ke direktori proyek: 
    cd nama-proyek
  3. Jalankan server lokal: 
    php artisan serve
  4. Buka browser dan akses http://localhost:8000.

Instalasi Versi Lain

Untuk menginstal versi Laravel tertentu, gunakan perintah berikut:

  • Laravel 10: composer create-project laravel/laravel:^10.0 nama-proyek
  • Laravel 9: composer create-project laravel/laravel:^9.0 nama-proyek
  • Laravel 8: composer create-project laravel/laravel:^8.0 nama-proyek
